# Retail Sales Performance Dashboard

This Power BI dashboard analyzes retail sales data to provide insights on
sales, profit, quantity, and customer segments.

## Tools Used
- Power BI
- Excel
- DAX

## Key Insights
- Sales by region and category
- Monthly sales & profit trends
- Segment-wise performance
